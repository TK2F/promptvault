> AIプロンプト管理用 Chrome拡張機能

ChatGPT / Gemini / Claude利用時に、プロンプトを効率的に「検索・再利用・編集」できるChrome拡張です。

---

## ✨ 機能

| 機能 | 説明 |
|------|------|
| 🔍 **リアルタイム検索** | タイトル・内容・タグを横断検索（大文字/小文字区別オプション付き） |
| 📌 **ピン留め** | よく使うプロンプトを常に上部に表示 |
| 🏷️ **複数タグ対応** | カンマ、スペース、タブで一括入力 |
| 🎯 **ドラッグ&ドロップ** | マウスで簡単に順序変更 |
| 🔖 **タグ/カテゴリフィルター** | タグやカテゴリをクリックで絞り込み（再クリックで解除） |
| 📊 **ソート順選択（7種類）** | カスタム/更新日時/登録日時/名前順（昇順・降順対応） |
| 📋 **ワンクリックコピー** | 即座にAIチャットに貼り付け |
| 🌐 **多言語対応** | 日本語/英語切り替え |
| 🌙 **ダーク/ライトモード** | OS設定に連動（手動切替も可） |
| 🎨 **グラスモーフィズムUI** | モダンで美しいガラス風デザイン |
| 📝 **Markdownプレビュー** | 構造化されたプロンプトを見やすく表示 |
| 💾 **オフライン対応** | ローカルストレージで完全オフライン動作 |
| 📤 **エクスポート/インポート** | JSON/CSV形式でバックアップ・移行（タグ/カテゴリフィルター対応） |

## 📦 インストール

### ソースからビルド

```bash
git clone https://github.com/YOUR_USERNAME/promptvault.git
cd promptvault
npm install
npm run build
```

1. Chrome で `chrome://extensions` を開く
2. 「**デベロッパーモード**」を有効化
3. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
4. `dist` フォルダを選択
5. ツールバーのアイコンをクリックしてサイドパネルを開く

## 🚀 使い方

### 基本操作

| 操作 | 方法 |
|------|------|
| サイドパネルを開く | ツールバーのアイコンをクリック |
| プロンプト追加 | 右下の「+」ボタン |
| 検索 | 検索バーにキーワード入力 |
| コピー | カードにホバー → コピーボタン |
| ピン留め | カードにホバー → ピンボタン |
| 並び替え | 左端のハンドルをドラッグ |

### キーボードショートカット

| キー | 機能 |
|------|------|
| `Esc` | 詳細画面を閉じる / 編集キャンセル |
| `Ctrl+S` | 編集内容を保存 |

## 📤 データ管理

### エクスポート
- **フォーマット**: JSON または CSV
- **カテゴリフィルター**: 指定カテゴリのみ出力可能
- **全件出力**: カテゴリ未選択時

### インポート
- **対応フォーマット**: JSON / CSV
- **追加モード**: 既存プロンプトに追加
- **置換モード**: 既存を削除して新規読み込み

### Danger Zone
- 全データ削除（「DELETE」入力で確認）
