# 使い方ガイド

## 画面構成

PromptVaultは以下の3つの画面で構成されています：

1. **リスト画面** - プロンプト一覧、検索、ピン留め
2. **詳細画面** - プロンプトの内容表示・編集
3. **設定画面** - テーマ、言語、データ管理

---

## 基本操作

### プロンプトを追加する

1. 右下の「**+**」ボタン（グロー効果付き）をクリック
2. 以下の情報を入力：
   - **名前**（必須）: プロンプトのタイトル
   - **カテゴリ**（任意）: 分類用ラベル
   - **タグ**（任意）: 複数のタグをカンマ、スペース、タブで区切って入力
   - **内容**（必須）: プロンプト本文
3. 「**追加**」をクリック

### プロンプトを検索する

検索バーにキーワードを入力すると、以下を対象にリアルタイム検索します：
- 名前
- 内容
- カテゴリ
- タグ

#### 大文字/小文字を区別

検索バーの下にある「**大文字/小文字を区別**」ボタンで切り替えられます。
- **ON**（青色）: 完全一致検索
- **OFF**（グレー）: 大文字小文字を無視

### プロンプトをコピーする

プロンプトカードにマウスを乗せると、右側にボタンが表示されます：
1. **コピーボタン**（📋）をクリック
2. クリップボードにコピーされます
3. ChatGPT/Gemini/Claudeに貼り付けて使用

### プロンプトをピン留めする

よく使うプロンプトをリストの上部に固定できます。

1. プロンプトカードにマウスを乗せる
2. **ピンボタン**（📌）をクリック
3. ピン留めセクションに表示されます

解除するには再度ピンボタンをクリックします。

### プロンプトの順番を変える

ドラッグ&ドロップで並び替えできます。

1. プロンプトカードの左側にあるハンドル（⠿）をドラッグ
2. 好きな位置にドロップ
3. 順序が自動保存されます

### プロンプトを編集する

1. プロンプトカードをクリックして詳細画面を開く
2. 右上の**編集ボタン**（✏️）をクリック
3. 内容を編集
4. **保存ボタン**（💾）をクリック、または `Ctrl+S`

### プロンプトを削除する

1. プロンプトカードにマウスを乗せる
2. **削除ボタン**（🗑️）をクリック
3. 確認ダイアログで「OK」をクリック

### 右クリックでプロンプトを追加する（v1.6）

Webページでテキストを選択して、右クリックから直接追加できます：

1. 任意のWebページでテキストをハイライト
2. **右クリック → 「PromptVaultに追加」**
3. 自動的にプロンプトとして保存されます
   - 名前: テキストの先頭30文字
   - カテゴリ・タグ: 未設定（後で編集可能）

**空行の処理設定**:
- 設定画面で「**空行を保持（1つまで）**」または「**空行を削除**」を選択可能
- デフォルトは「空行を保持（1つまで）」

### 未設定プロンプトをフィルターする（v1.6）

右クリックで追加したプロンプトはカテゴリ・タグが未設定です。
これらを一括で管理するためのフィルター機能があります：

1. 検索バーの右にある「**◯**」ボタン（オレンジ色の円）をクリック
2. カテゴリ・タグが未設定のプロンプトのみ表示されます
3. 再度クリックで解除

### クイックガイドを表示する（v1.6）

操作方法が分からないときは、アプリ内ガイドを参照できます：

1. ヘッダーの「**?**」ボタンをクリック
2. 使い方ガイドがポップアップ表示されます
3. 言語設定に応じて日本語/英語で表示

## フィルタリング

### タグで絞り込み

プロンプトカードのタグをクリックすると、同じタグを持つプロンプトのみ表示されます。

- **緑色チップ**: タグフィルター
- **クリックでフィルター適用**
- **再クリックでフィルター解除**（v1.6.1）

### カテゴリで絞り込み

プロンプトカードのカテゴリをクリックすると、同じカテゴリのプロンプトのみ表示されます。

- **青色チップ**: カテゴリフィルター
- **クリックでフィルター適用**
- **再クリックでフィルター解除**（v1.6.1）

### 詳細画面でのフィルター（v1.6.1）

詳細画面でカテゴリやタグをクリックすると：

1. フィルターが適用／解除される
2. 自動的にリスト画面に戻る

### 複合フィルター

タグ + カテゴリ + キーワード検索を組み合わせて使用できます。

### フィルターのクリア

- 検索バー下のフィルターバッジの「×」をクリック
- または「すべてクリア」リンクをクリック

## 設定

歯車アイコン（⚙️）をクリックして設定画面を開きます。

### 言語

- 🇯🇵 **日本語**
- 🇺🇸 **English**

### テーマ

- ☀️ **ライト** - 明るい背景（グラデーション）
- 🌙 **ダーク** - 暗い背景（グラスモーフィズム）
- 🖥️ **自動** - OSの設定に合わせる

### フォントサイズ

- **小** - コンパクト表示
- **中** - 標準
- **大** - 大きめの文字

### 空行の処理（v1.6）

右クリック追加時の空行の扱いを設定できます：

- **空行を保持（1つまで）** - 段落構造を維持（デフォルト）
- **空行を削除（コンパクト）** - すべての空行を削除

### 並び替え順（v1.6.1）

プロンプト一覧の並び順を選択できます。ヘッダーの↑↓ボタンまたは設定から変更可能です：

| 項目 | 説明 |
|------|------|
| **カスタム順** | ドラッグ＆ドロップで設定した順序（デフォルト） |
| **更新日時（新→古）** | 最近更新したプロンプトが上 |
| **更新日時（古→新）** | 古い更新順 |
| **登録日時（新→古）** | 最近追加したプロンプトが上 |
| **登録日時（古→新）** | 古い追加順 |
| **名前順（A→Z）** | アルファベット・あいうえお順 |
| **名前順（Z→A）** | 逆順 |

## データ管理

### エクスポート

プロンプトをファイルに出力できます：

1. 設定画面 → **エクスポート**ボタンをクリック
2. フォーマットを選択:
   - **JSON形式** - バックアップ・移行用
   - **CSV形式** - Excel・スプレッドシート用
3. フィルター（任意）:
   - **カテゴリ**: 特定カテゴリのみ出力
   - **タグ**: 特定タグを持つプロンプトのみ出力
   - **選択なし = 全件出力**
4. **エクスポート**ボタンをクリック

### インポート

別のPCからのお引っ越しや、バックアップの復元ができます：

1. 設定画面 → **インポート (JSON/CSV)** ボタンをクリック
2. ファイルを選択
3. インポート方法を選択:
   - **追加**: 既存のプロンプトに追加
   - **置換**: 既存を全て削除してから読み込み
4. **インポート**ボタンをクリック

**注意**: 置換モードでは既存のプロンプトが全て削除されます

### 全データ削除（Danger Zone）

すべてのデータを削除したい場合：

1. 設定画面 → 一番下の **Danger Zone**
2. **全データを削除**ボタンをクリック
3. 確認ダイアログで「DELETE」と入力
4. **削除**ボタンをクリック

**警告**: この操作は取り消せません

## データの永続性

| 状況 | プロンプトへの影響 |
|------|------------------|
| Chrome再起動 | ✅ 保持される |
| Chrome更新 | ✅ 保持される |
| キャッシュクリア | ✅ 保持される |
| 拡張機能の削除 | ⚠️ 削除される |

**推奨**: 定期的にエクスポートでバックアップを取りましょう

## キーボードショートカット

| キー | 機能 |
|------|------|
| `Esc` | 詳細画面を閉じる / 編集キャンセル |
| `Ctrl+S` | 編集内容を保存 |

## ヒント

- **最近使用セクション**: 直近でコピーしたプロンプトが表示されます
- **タグ入力**: Enter、Tab、カンマで確定できます
- **Markdown対応**: プロンプト内容はMarkdownとして表示されます
- **パフォーマンス**: 多数のプロンプトを管理しても快適に動作します
